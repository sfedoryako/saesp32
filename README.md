# Smart Speaker на ESP32-S3 (Голосовой помощник "Алёна")

Умная колонка на базе ESP32-S3 с интеграцией Yandex SpeechKit для распознавания и синтеза речи.

## 🎯 Возможности

### Голосовые команды
- **Активация**: "Алёна" + команда
- **Погода**: "Алёна, какая погода в Воронеже?"
- **Время**: "Алёна, сколько сейчас времени?"
- **Курсы валют**: "Алёна, какой курс биткоина?"
- **Радио**: "Алёна, включи радио Рекорд"
- **Управление**: громче/тише, стоп/молчанка

### Радиостанции
- **Рекорд** - Радио Рекорд
- **Европа Плюс** - Europa Plus
- **Наше Радио** - Наше Радио

### Веб-интерфейс
- Управление через браузер
- Wi-Fi Manager для подключения к сети
- Мониторинг команд и статуса
- Настройка громкости
- Спящий режим

## 🔧 Аппаратные требования

### Основные компоненты
- **ESP32-S3** - основной микроконтроллер
- **I2S микрофон** - для записи голоса
- **I2S ЦАП PCM5102A** - для аудио вывода
- **Динамики/наушники** - для воспроизведения

### Схема подключения

#### I2S Микрофон (RX)
```
GPIO 17 - SCK (Serial Clock)
GPIO 18 - WS (Word Select) 
GPIO 16 - SD (Serial Data)
```

#### I2S ЦАП PCM5102A (TX)
```
GPIO 19 - BCK (Bit Clock)
GPIO 21 - LRCK (Left/Right Clock)
GPIO 10 - DIN (Data Input)
```

#### Индикация
```
GPIO 2 - LED (встроенный светодиод)
```

## 📦 Установка и настройка

### 1. Arduino IDE
1. Установите Arduino IDE
2. Добавьте поддержку ESP32:
   - File → Preferences → Additional Board Manager URLs
   - Добавьте: `https://raw.githubusercontent.com/espressif/arduino-esp32/gh-pages/package_esp32_index.json`
3. Tools → Board → Boards Manager → поиск "ESP32" → установка

### 2. Библиотеки
Установите через Library Manager:
```
- WiFi (встроенная)
- WebServer (встроенная) 
- HTTPClient (встроенная)
- WiFiClientSecure (встроенная)
- ArduinoJson by Benoit Blanchon
- AudioTools by Phil Schatzmann
- SPIFFS (встроенная)
```

### 3. Загрузка файлов
1. Откройте `SmartSpeaker.ino` в Arduino IDE
2. Загрузите данные SPIFFS:
   - Tools → ESP32 Sketch Data Upload
   - Это загрузит файлы из папки `data/` в память ESP32

### 4. API ключи
Обновите в коде:
```cpp
const char* YANDEX_API_KEY = "ваш_ключ_yandex";
const char* YANDEX_FOLDER_ID = "ваш_folder_id";
const char* OPENWEATHER_API_KEY = "ваш_ключ_openweather";
```

## 🚀 Первый запуск

### 1. Подключение к Wi-Fi
1. После прошивки ESP32 создаст точку доступа "SmartSpeaker"
2. Подключитесь с паролем: `12345678`
3. Откройте браузер и перейдите на `192.168.4.1`
4. Выберите вашу домашнюю Wi-Fi сеть и введите пароль
5. Устройство перезагрузится и подключится к домашней сети

### 2. Использование
1. Дождитесь подключения к Wi-Fi (звуковой сигнал)
2. Произнесите "Алёна" + команду
3. Устройство ответит голосом или выполнит действие

## 🌐 Веб-интерфейс

После подключения к Wi-Fi устройство будет доступно по IP адресу в вашей сети:
- Отображение статуса подключения
- Последняя выполненная команда
- Регулировка громкости
- Управление спящим режимом
- Быстрые команды
- Монитор активности

## 🔧 Технические детали

### Архитектура
- **FreeRTOS** - многозадачность
- **I2S** - высококачественное аудио
- **SPIFFS** - файловая система для веб-интерфейса
- **WiFiClientSecure** - HTTPS соединения с API

### Память
- **PSRAM** - для обработки аудио
- **Flash** - хранение программы и SPIFFS
- **RAM** - буферы I2S и переменные

### Производительность
- **Частота дискретизации**: 44.1 кГц (вывод), 16 кГц (ввод)
- **Разрядность**: 16 бит
- **Латентность**: ~3 секунды на команду
- **Пороговая амплитуда**: 50 (настраивается)

```cpp

```

## 🐛 Отладка

### Последовательный монитор
- Скорость: 115200 бод
- Отображает статус инициализации
- Логи команд и ошибок
- Мониторинг памяти

### Частые проблемы
1. **Нет звука** - проверьте подключение I2S ЦАП
2. **Не записывает** - проверьте I2S микрофон
3. **Не подключается к Wi-Fi** - сбросьте настройки через веб-интерфейс
4. **Большое потребление памяти** - следите за логами heap

## 📋 API

### Yandex SpeechKit
- **STT**: `stt.api.cloud.yandex.net`
- **TTS**: `tts.api.cloud.yandex.net`
- Формат: LPCM, 16 кГц, 16 бит

### External APIs
- **OpenWeatherMap**: погода
- **CoinGecko**: курсы криптовалют
- **WorldTimeAPI**: время

## 🔒 Безопасность

- HTTPS соединения для всех API
- SSL сертификат в SPIFFS
- Нет хранения паролей в открытом виде
- Таймауты для HTTP запросов
