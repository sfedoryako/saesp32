# 🎤 ESP32-S3 Microphone Test

Скрипт для тестирования микрофона ESP32-S3 умной колонки. Позволяет записывать звук с микрофона и воспроизводить его в реальном времени через динамики, чтобы вы могли услышать то, что подает микрофон.

## ✨ Возможности

- 🎙️ **Запись аудио в реальном времени** с I2S микрофона
- 🔊 **Воспроизведение** записанного звука через I2S динамики
- 🎚️ **Настраиваемое усиление** (0.1x - 10x)
- 📊 **Мониторинг уровня сигнала** в реальном времени
- 🌐 **Веб-интерфейс** для удобного управления
- 💡 **LED индикация** активности микрофона
- 📈 **Визуальный индикатор** уровня звука

## 🔧 Технические характеристики

- **Частота дискретизации**: 16000 Гц
- **Разрядность**: 16 бит
- **Вход**: Моно (микрофон)
- **Выход**: Стерео (динамики)
- **Задержка**: Минимальная (real-time)

## 🚀 Как использовать

### 1. Выбор версии скрипта

Доступны две версии:
- **`MicrophoneTest.ino`** - полная версия с эмодзи и Unicode символами
- **`MicrophoneTest_ASCII.ino`** - совместимая версия только с ASCII символами

⚠️ **При ошибках компиляции используйте ASCII версию!**

### 2. Загрузка скрипта

1. Откройте Arduino IDE
2. Загрузите один из файлов на ESP32-S3:
   - `MicrophoneTest.ino` (предпочтительно)
   - `MicrophoneTest_ASCII.ino` (если есть проблемы с компиляцией)
3. Откройте Serial Monitor (115200 baud) для мониторинга

### 2. Подключение к устройству

После загрузки ESP32-S3 создаст точку доступа WiFi:

- **SSID**: `ESP32_MicTest`
- **Пароль**: `12345678`
- **IP адрес**: `192.168.4.1`

### 3. Веб-интерфейс

1. Подключитесь к WiFi сети `ESP32_MicTest`
2. Откройте браузер и перейдите на `http://192.168.4.1`
3. Вы увидите веб-интерфейс с панелью управления

### 4. Тестирование микрофона

1. Нажмите кнопку **"▶ Начать тест"**
2. Говорите в микрофон
3. Вы должны услышать свой голос через динамики
4. Наблюдайте за уровнем сигнала на индикаторе
5. При необходимости регулируйте усиление
6. Нажмите **"⏹ Остановить тест"** для завершения

## 🎛️ Настройки

### Усиление микрофона

- **0.1x - 1.0x**: Для очень чувствительных микрофонов
- **1.0x - 3.0x**: Стандартные настройки
- **3.0x - 10x**: Для слабых сигналов

### Индикация

- **LED мигает**: Микрофон принимает звук
- **Индикатор уровня**: Показывает амплитуду сигнала
- **Числовые значения**: Текущая и максимальная амплитуда

## 🛠️ Подключение оборудования

### Микрофон (I2S)
- **BCK**: GPIO 17
- **WS/LRC**: GPIO 18  
- **DATA**: GPIO 16
- **VCC**: 3.3V
- **GND**: GND

### Динамики/ЦАП (I2S)
- **BCK**: GPIO 19
- **LRC**: GPIO 21
- **DATA**: GPIO 10

### LED индикация
- **LED**: GPIO 2 (встроенный)

## 🔍 Диагностика проблем

### Нет звука
- Проверьте подключение динамиков к пинам I2S TX
- Убедитесь что динамики включены и работают
- Попробуйте увеличить усиление

### Тихий звук
- Увеличьте усиление в веб-интерфейсе
- Проверьте расстояние до микрофона
- Убедитесь что микрофон правильно подключен

### Искажения
- Уменьшите усиление
- Проверьте питание микрофона
- Говорите тише или отойдите от микрофона

### Низкая амплитуда
- Проверьте подключение микрофона к пинам I2S RX
- Убедитесь что микрофон получает питание
- Проверьте правильность пинов BCK, WS, DATA

## 🔧 Технические детали

### Архитектура
- **Ядро 0**: Веб-сервер, мониторинг
- **Ядро 1**: Захват и воспроизведение аудио
- **Синхронизация**: Мьютексы для thread-safe операций

### Производительность
- **Размер буфера**: 512 сэмплов
- **Количество буферов**: 8
- **Задержка**: ~32ms (минимальная)
- **CPU загрузка**: ~30-40%

### Память
- **RAM использование**: ~50KB
- **PSRAM**: Не требуется
- **Flash**: ~300KB

3. Попробуйте разные настройки усиления
4. Перезагрузите ESP32-S3 если есть проблемы
